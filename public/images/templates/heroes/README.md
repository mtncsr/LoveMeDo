# Hero Placeholder Images

Place the following 5 hero placeholder images in this folder:

1. `birthday-kids-hero.jpg` (or .png, .webp, etc.) - For birthday-kids template (Screen 2: Hero)
2. `one-screen-hero.jpg` (or .png, .webp, etc.) - For one-screen template (Screen 1: Main - fullscreen)
3. `baby-birth-hero.jpg` (or .png, .webp, etc.) - For baby-birth template (Screen 2: Details)
4. `memorial-hero.jpg` (or .png, .webp, etc.) - For memorial template (Screen 2: Portrait)
5. `cny-hero.jpg` (or .png, .webp, etc.) - For cny template (Screen 2: Zodiac Hero)

**Note**: You can use any browser-supported image format (JPG, PNG, WebP, etc.). Just ensure the file extension matches the actual format.
