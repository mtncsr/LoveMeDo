# Video Placeholder

Place the following video placeholder file in this folder:

- `video-placeholder.mp4` - This single video will be reused for all video elements across all templates

**Note**: This video placeholder will be used for video elements in all templates:
- birthday-kids
- birthday-partner
- anniversary-timeline
- graduation
- new-year

**Format**: MP4 (H.264 codec recommended for best browser compatibility)
