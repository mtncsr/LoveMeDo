1. `gallery-1.jpg` (or .png, .webp, etc.)
2. `gallery-2.jpg` (or .png, .webp, etc.)
3. `gallery-3.jpg` (or .png, .webp, etc.)
4. `gallery-4.jpg` (or .png, .webp, etc.)
